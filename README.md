# Dotfiles

## Requirements
- [fish shell](https://fishshell.com/docs/current/)
- [neovim](https://neovim.io/doc/user/)
- [tree](https://linux.die.net/man/1/tree)
- [ag](https://github.com/ggreer/the_silver_searcher)
- [fzf](https://github.com/junegunn/fzf)

## Install

```
cp -r .config ~/.config
nvim +PlugInstall
```
